# TOEIC 500→730 弱点単語アプリ

## 収録語数
442語

## 使い方
1. このフォルダの中身をGitHubのリポジトリ直下に置く
2. Settings → Pages → Deploy from a branch → main / root
3. 公開URLをSafariで開く
4. 共有 → ホーム画面に追加

## 機能
- 4択クイズ
- A/B/C/Dの正解位置ランダム
- 即時判定
- 日本語訳・例文・覚えるポイント表示
- 間違えた語ほど出やすい
- 3回連続正解で卒業、★重要語は4回連続正解で卒業
- Part 1 / Part 5-7 / ★重要 / 間違いだけ / 卒業語モード
- localStorage保存
- 学習記録のJSONエクスポート/インポート

## 注意
学習記録は端末内のブラウザに保存されます。SafariのWebサイトデータを消すと消える可能性があります。
